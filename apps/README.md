# Applications

## Active Product

[`current/`](current/) is iteration 07 (V7), the only active development target.

## Historical Milestones

[`history/`](history/) contains six independently runnable snapshots. They are
ordered numerically and named for the main design change they introduced.
Historical code is preserved so product and engineering decisions can be
compared directly; the current app does not import it.

Use the root [project history](../docs/PROJECT_HISTORY.md) for the narrative and
each app's README for its original capabilities, architecture, commands, and
limitations.
