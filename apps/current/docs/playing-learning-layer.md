# V8 Playing and Creative Learning Layer

Every core relationship moves through a repeatable arc:

```text
Hear → predict → imitate → locate → name → compare → vary → create → transfer → reflect
```

Activities record attempts, assistance, outcomes, elapsed time, and musical
context. A hint or reveal can help the learner continue, but it does not count as
independent mastery. Secure evidence requires successful unassisted work on two
different days; transfer-ready evidence also requires a changed key, neck region,
tempo, instrument, or creative context.

## Sketchbook

Creative work follows Capture, Understand, Vary, Arrange, Record, Compare,
Revise, and Finish. Sketches persist automatically and retain intention, tempo,
metre, key or no declared key, chord events and harmonic rhythm, rhythm text,
notes, workflow status, takes, and revisions. Transformation prompts explain
which musical relationship changes without grading the result as right or wrong.

## Privacy

Recordings stay in browser storage and are included in the local versioned
archive export. There is no account, upload, server, or analytics layer.
